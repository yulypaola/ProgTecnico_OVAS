Estructuras Condicionales 🔀

Permiten que el algoritmo tome decisiones. Según una condición, el flujo continúa por un camino u otro.

Unidad 1 · Tema 4 — Condicionales

¿Qué son y para qué sirven?

Una condicional evalúa una expresión booleana (verdadero/falso) y ejecuta uno u otro bloque de instrucciones según el resultado. Son clave para validar datos y controlar el flujo de tu programa.

Recuerda: primero analiza el problema (Tema 2) y define E–P–S (Tema 3) antes de agregar decisiones.

Tipos de condicional 🌿

Simple

SI (llueve) ENTONCES
  llevar paraguas
FIN SI

Doble

SI (edad >= 18) ENTONCES
  "Mayor de edad"
SINO
  "Menor de edad"
FIN SI

Múltiple

SEGÚN (día)
  CASO "Lun": "Inicio"
  CASO "Sab": "Descanso"
  OTRO: "Clase"
FIN SEGÚN

Operadores 🔗

Relacionales

  • <, >, <=, >=, ==, !=
  • Ej.: nota >= 3.0

Lógicos

  • AND (y), OR (o), NOT (no)
  • Ej.: edad >= 18 AND documento == válido

Ejemplos 📄

Ejemplo 1: Par/Impar

INICIO
  leer n
  SI (n % 2 == 0) ENTONCES
    mostrar "Par"
  SINO
    mostrar "Impar"
FIN

Ejemplo 2: Aprobado/Reprobado

INICIO
  leer nota
  SI (nota >= 3.0) ENTONCES
    mostrar "Aprueba"
  SINO
    mostrar "Reprueba"
FIN

¡Practica! ✍️

1. Mayoría de edad

Lee edad y muestra si es mayor o menor de edad.

2. Descuento en tienda

Si compra >= $100.000 aplica 10% de descuento, si no, precio normal.

3. Clasificación de temperatura

Según °C: < 15 “Frío”; 15–25 “Agradable”; > 25 “Caliente”.
