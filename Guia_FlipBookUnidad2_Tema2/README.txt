Cómo abrir:
1) Descarga y descomprime la carpeta flipbook.
2) Abre index.html en tu navegador.

Publicar en GitHub Pages:
1) Crea un repositorio y sube todo el contenido de 'flipbook/'.
2) En Settings → Pages, selecciona branch 'main' y carpeta raíz.
3) Guarda; la URL pública se generará automáticamente.
