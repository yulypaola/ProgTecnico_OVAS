HTML

significa HyperText Markup Language, es decir, Lenguaje de Marcado de Hipertexto.Es el lenguaje base para crear páginas web, y permite estructurar el contenido que vemos en Internet, como títulos, párrafos, imágenes, enlaces, formularios, botones, entre otros.

HTML no es un lenguaje de programación (no tiene lógica como condicionales o bucles), sino un lenguaje de estructura que usa etiquetas (tags) para indicar qué es cada elemento dentro de una página web.
